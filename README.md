WanderLust – Travel Listing Platform

A full-stack web application that allows users to explore, create, and manage travel property listings. 
Users can sign up, log in, post listings, upload images, and leave reviews on listings.

FEATURES
User Authentication (Signup / Login / Logout)
*  Create, Edit, Delete Listings
*  Upload Images for Listings
*  Add Reviews to Listings
*  Authorization (Only owners can edit/delete listings)
*  Flash messages for alerts
*  Responsive UI with Bootstrap
*  Deployed using Render

Tech Stack
-> Frontend
      *  HTML
      *  CSS
      *  Bootstrap
      *  EJS
->  Backend
      *  Node.js
      *  Express.js
->  Database
      * MongoDB Atlas
      * Mongoose
->  Other Tools
